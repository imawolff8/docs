---
title: "更新 NextNav"
sidebarTitle: "更新"
description: "定期更新 NextNav 到最新版本，可以让您的 NextNav 实例 Bug 更少、功能更强。"
---

<Steps>
  <Step title="获取 nextnav image 的最新版本号">
    ```bash
    curl "https://registry.887676.xyz/v2/beyondim/nextnav/tags/list"
    {"name":"beyondim/nextnav","tags":["1.0.0", "1.0.1", "1.0.2"],"latest":"1.0.2"}
    ```
  </Step>
  <Step title="下载最新版 nextnav image">
    ```bash
    # 请将 VERSION 修改为上面获取到的最新版本号，比如 1.0.2
    docker pull registry.887676.xyz/beyondim/nextnav:VERSION
    ```
  </Step>
  <Step title="更新">
    ```bash
    cd nextnav_directory
    docker compose down
    docker compose up
    ```
  </Step>
</Steps>
