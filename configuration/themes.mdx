NextNav ships with multiple built-in themes that let you change the entire look and feel of your bookmark dashboard without touching any code. Themes control everything from layout structure and color palette to interactive features like drag-and-drop sorting and inline editing.

## Switching Themes

<Steps>
  <Step title="Log in as admin">
    Navigate to your NextNav URL and sign in with your admin credentials.
  </Step>
  <Step title="Open the Theme selector">
    Go to **Admin Panel** → **Settings** → **Theme** to find the theme dropdown.
  </Step>
  <Step title="Choose a theme">
    Select a theme from the dropdown list. A preview description appears for each option to help you decide.
  </Step>
  <Step title="Save your selection">
    Click **Save**. Your dashboard updates immediately — no reload required.
  </Step>
</Steps>

<Note>
  Theme preferences are stored per-site, not per-user. Changing the theme affects all visitors to your NextNav instance, including guests.
</Note>

## Available Themes

<CardGroup cols={2}>
  <Card title="default" icon="layout-grid">
    A clean, minimal layout focused on simplicity and speed. Bookmarks are presented in a straightforward grid without additional UI chrome. Best suited for users who prefer a distraction-free experience and manage their bookmarks primarily through the Admin Panel.
  </Card>
  <Card title="default2" icon="sparkles">
    An enhanced layout that adds front-end editing, drag-and-drop sorting, and a right-click context menu directly on the dashboard. Recommended for most users — you can reorganize, add, and edit bookmarks without ever opening the Admin Panel.
  </Card>
</CardGroup>

<Tip>
  The `default2` theme unlocks additional features like inline bookmark editing and drag-and-drop sorting without going to the admin panel. If you manage your bookmarks frequently, this theme significantly speeds up your workflow.
</Tip>
